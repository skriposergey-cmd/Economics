# Economics# Rewise Studio — Економіка

Калькулятор собівартості та маржі послуг. Частина екосистеми Rewise Studio (поруч із [Прайсом](https://rewise-studio.github.io/Price/) і [CRM](https://rewise-studio.github.io/crm/)).

🔗 Сайт: https://rewise-studio.github.io/Economics/

## Що тут

Один файл `index.html` — статична сторінка, без збірки. Захищена паролем (той самий, що в CRM — спільна сесія в межах `rewise-studio.github.io`).

Дані не зберігаються в репозиторії — все живе в Google Sheets:
- Таблиця: `1kDJ893PFa_DTNKt-r8wQES_JFXbJSDNU8niRLz70Aao`
- Листи: **Прайс** (тільки читання — список послуг і ціни), **Матеріали** (каталог матеріалів), **Економіка** (збережені розрахунки)

Читання — напряму з Google Sheets (gviz JSON, без бекенда). Запис («Додати матеріал», «Зберегти розрахунок») — через Flask API на Railway.

## Як це працює

- **Матеріали** — каталог на кшталт «Фарба Fenice, 500 г, 450 ₴» вноситься один раз, ціна за одиницю рахується автоматично, далі обирається зі списку при розрахунку послуги.
- **Розрахунок послуги** — обираєш послугу з Прайсу (або вводиш вручну), додаєш матеріали, норма-год і накладні (опціонально) → бачиш собівартість і маржу.
- **Курс $** — підтягується наживо з PrivatBank (готівковий курс) і записується в момент збереження розрахунку, щоб з часом бачити, наскільки курс «втік» від ціни.

## Якщо потрібно щось змінити

- **Бекенд** (`crm_api.py`, ендпоінти `/material` і `/economics`) — лежить у репозиторії [`Rewise-Studio/Price`](https://github.com/Rewise-Studio/Price), деплоїться на Railway автоматично після коміту.
- **Структура листів Google Sheets** — якщо додаєш нову колонку, не забувай синхронно поправити і `crm_api.py` (порядок колонок у `row_data`), і читання в `index.html` (функції `loadMaterials` / `loadEconomics`, індекси колонок).
